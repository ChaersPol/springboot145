**3.1 可行性分析**

**3.1.3 操作可行性**

小学生身体素质测评管理系统的具体实现，本身参考人类的正常操作逻辑，把常用的操作习惯当做主要的导航实现，可以让使用者更快速的理解并且上手操作，实现符合逻辑的操作流程是操作可行性的具体体现。
以上就是从不同的角度来分析，确保了小学生身体素质测评管理系统的正常开展。
